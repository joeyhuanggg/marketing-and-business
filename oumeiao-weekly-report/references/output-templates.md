# Output Templates

Use these templates as the default output structures.

## Half-Week Template

1. Title
2. Data scope
3. Overall progress
4. This half-week's visible changes
5. Weak channels
6. Next actions before week close

Suggested tables:
- overall progress table
- weak channels table

## Weekly Template

1. Title
2. Data scope
3. Overall progress
4. Channel structure
5. Weak channels
6. Better channels
7. Next-week actions

Suggested tables:
- overall progress table
- channel performance table
- weak channels table

## Monthly Template

1. Title
2. Data scope
3. Month result summary
4. Structural gains
5. Structural losses
6. Budget efficiency review
7. Next-month actions

Suggested tables:
- month result table
- structural channel review table
- low-efficiency channels table
