# Metric Taxonomy

Use this taxonomy for `海外商务-链路达成数据`.

- `主投_例子数`
- `主投_例子达成率`
- `主投_约课数达成`
- `主投_约课数MTD达成率`
- `主投GMV情况 > 滚动GMV达成`
- `主投GMV情况 > 滚动GMV达成率`
- `合计GMV情况 > 滚动GMV达成`
- `合计GMV情况 > 滚动GMV达成率`
- `ROI2情况 > 滚动ROI2达成`

## Diagnosis Shortcuts

- Spend high + example achievement low: top-of-funnel or channel quality problem
- Examples present + appointments weak: example-to-appointment problem
- Appointments present + attendance weak: attendance problem
- Attendance present + GMV zero: conversion problem
- GMV present + ROI2 weak: efficiency or ASP problem
- Main-invest okay + total GMV weak: inspect secondary-invest contribution
